# Cable Design Validator

AI-powered validation system for **IEC 60502-1 low-voltage cable designs**.  
This application validates cable design inputs against IEC standards and returns structured compliance results with confidence scoring.

## Features

-   IEC 60502-1 compliance validation
    
-   Field-level status: **PASS / WARN / FAIL**
    
-   Confidence scoring
    
-   FastAPI backend
    
-   Streamlit frontend
    
-   LLM-assisted rule interpretation
    
-   Modular, extensible architecture
    

  

## Installation

### Prerequisites

-   Python **3.11+**
    
-   **uv** package manager
    
-   Gemini API Key
